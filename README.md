# Monuments etrangers 14-18 en Suisse
Ce dépôt GitHub contient uniquement la galerie des photos des monuments étrangers de la première Guerre Mondiale en Suisse.

Il ne s'agit pas de photos professionnelles mais surtout de photographies permettant la lecture des noms mentionnés sur chaque monument.
Il ne s'agit également pas de la liste complète du corpus car pour certains monuments, notamment plusieurs plaques, un déplacement n'a pas été nécessaire.

Les photos sont de haute qualité, permettant de zoomer pour lire les noms avec satisfaction, mais cela  implique donc un poids élévé.
